# Telegram OSINT Image Analyzer Bot
*Универсальный Telegram бот для глубокого анализа изображений: извлечение метаданных (EXIF, GPS), вычисление хешей, AI-анализ с Gemini и очистка фотографий от конфиденциальной информации.*

## Что это?
Этот Telegram-бот предоставляет мощный инструмент для OSINT (Open Source Intelligence) анализа изображений. Он автоматизирует процесс извлечения ценной информации из фотографий, помогая исследователям и пентестерам быстро получать данные.

**Основные функции:**
*   **Извлечение EXIF-метаданных:** Получение подробной информации о камере, настройках съемки, дате и времени.
*   **Геолокация (GPS):** Извлечение GPS-координат из метаданных и генерация прямой ссылки на Google Maps.
*   **Вычисление хешей:** Расчет MD5 и SHA256 хешей файла для идентификации и проверки целостности.
*   **AI-анализ с Google Gemini:** Отправка изображения в Google Gemini для расширенного визуального анализа (описание места, объектов, текста на изображении).
*   **Очистка метаданных:** Создание копии изображения, полностью лишенной EXIF-метаданных, для анонимной публикации.

> [!INFO] **Зачем это нужно?**
> Пентестеры и исследователи безопасности часто используют OSINT для сбора информации о целях. Метаданные изображений могут содержать конфиденциальные данные, такие как GPS-координаты, модель устройства, серийные номера и даже личные данные. Этот бот помогает быстро извлечь эту информацию или, наоборот, очистить ее перед публикацией.

## Как установить?

### Необходимые компоненты
Убедитесь, что у вас установлен Python 3.x и менеджер пакетов `pip`.

```bash
python3 --version
pip3 --version
```

### Загрузка кода
Загрузите файл `osint_bot.py` на ваш сервер (например, Raspberry Pi 5, VPS или локальный Linux-машину).

### Установка зависимостей
Установите все необходимые Python-библиотеки, используя `pip`:

```bash
pip install pyTelegramBotAPI Pillow google-genai
```

## Как настроить?

### Получение Telegram Bot Token
1.  Откройте Telegram и найдите пользователя `@BotFather`.
2.  Отправьте ему команду `/newbot`.
3.  Следуйте инструкциям, чтобы выбрать имя и юзернейм для вашего бота.
4.  BotFather выдаст вам `HTTP API Token`. Скопируйте его.

### Получение Google Gemini API Key
1.  Перейдите на сайт [Google AI Studio](https://aistudio.google.com/).
2.  Войдите в свой Google аккаунт.
3.  Создайте новый API-ключ.
4.  Скопируйте полученный ключ.

> [!WARNING] **Безопасность API-ключей!**
> Никогда не публикуйте свои API-ключи в открытом доступе (например, в публичных репозиториях GitHub). Это может привести к несанкционированному использованию ваших ресурсов и финансовым потерям.

### Настройка файла `osint_bot.py`
Откройте файл `osint_bot.py` в текстовом редакторе и найдите секцию `КОНФИГУРАЦИЯ`.
Замените плейсхолдеры своими токенами:

```python
# ================= КОНФИГУРАЦИЯ =================
# Вставь сюда токен от BotFather
API_TOKEN = 'ТУТ_ТВОЙ_ТОКЕН_TELEGRAM' # <--- Вставьте сюда ваш токен

# Вставь сюда API ключ от Google (https://aistudio.google.com/)
GOOGLE_API_KEY = 'ТУТ_ТВОЙ_API_KEY_GOOGLE' # <--- Вставьте сюда ваш ключ
# ================================================
```

> [!TIP]
> Если вы не планируете использовать AI-анализ, можете оставить `GOOGLE_API_KEY` как есть. Бот будет работать без этой функции.

## Запуск бота
После установки зависимостей и настройки API-ключей, вы можете запустить бота:

```bash
python3 main.py
```

Бот начнет опрашивать сервера Telegram и будет готов к работе. Для запуска в фоновом режиме используйте `nohup` или `screen`/`tmux`.

```bash
nohup python3 main.py 
```

## Использование бота
1.  Найдите вашего бота в Telegram.
2.  Отправьте команду `/start`, чтобы получить приветственное сообщение и описание функций.
3.  **Для анализа фото:** Отправьте изображение как **Файл (Document)**. Это критически важно, так как при отправке как "Фото" Telegram сжимает изображение и удаляет большинство метаданных.
4.  Бот пришлет вам:
    *   Сообщение с EXIF-данными, GPS-координатами (если есть) и хешами.
    *   Текстовый файл с подробным AI-анализом изображения от Gemini.
    *   Копию вашего фото, полностью очищенную от метаданных.
